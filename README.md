# Dancing Box   
 > [PlayGame](https://junghc.github.io/dancingBox/)

### 參考:
  - 《跳舞的線》Dancing Line 遊戲

### 遊戲說明:
  - 按Enter遊戲開始
  - 按空白鍵變換方向，當方塊超出邊界及失敗
  
### 功能:
  - 單人模式: 地圖隨音樂改變挑戰抵達終點
  
### 使用技術:
  - 基本: HTML5, CSS, Javascript
  - 進階: Babylon.js
  
### 網站架構:
  - 遊戲主要以 **babylon.js** 為主
  - 透過 **CSS+jQuery** 完成開始和結束畫面
  
### 本網站的特色與優點:
  - 操作簡單
  - 音樂旋律、節奏為過關關鍵

### 分工:
